# Global IoT Project

A responsive and interactive landing page built with **Next.js** for the **IoT Lab at KIIT**. This project demonstrates a clean UI, modular components, and a modern layout for showcasing IoT services and updates.

### Key Features

- Responsive Hero section with interactive elements.  
- Sections for About, Services, News, Testimonials, and Why Us.  
- Modular component structure for maintainability.  
- Clean and modern UI for professional presentation.  
- Footer and header designed for smooth navigation.
